## Installation and Testing

Install fastapi dependencies and run

```bash
    pip install fastapi uvicorn[standard]
    cd backend
    uvicorn main:app --reload
```

Install vue js dependencies and run

```bash
    cd frontend
    npm install
    npm run dev
```
